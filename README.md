# Reminder application

Reminder application based on iOS system with ReactJS.

## Features

-   Add new reminder
-   Add new list
-   Color variation for list customization
-   Delete list
-   Delete reminder
-   Check reminder
-   Check all sticky notes
-   Add reminder dynamically
